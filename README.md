## Overview

This is the personal portfolio website. Built with modern web technologies, it emphasizes
accessibility, multilingual support, and responsive design.

## Features

- **Multilingual**: Supports English, Russian, and Ukrainian via
  [ngx-translate](https://github.com/ngx-translate/core).
- **Responsive Design**: Looks great on desktop and mobile devices.
- **Accessibility**: Semantic markup and ARIA attributes for screen readers.
- **Modern Stack**: Angular 20+, Angular Material, SCSS, TypeScript.
- **Project Showcase**: Interactive list of open-source and personal projects.
- **Protected Resume**: Secure access to resume in multiple languages, with download/export
  functionality.
- **Contact Section**: Quick access to professional and social contacts, with copy-to-clipboard
  support.
